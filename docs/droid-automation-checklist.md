# Droid Automation & Recurring Tasks – Phase 1

## Data Engineering Droid
- [ ] Cron/scheduler health check (options, political, calendars).
- [ ] Data Quality report (nulls, latency) → share weekly.
- [ ] Kafka/SQS consumer lag monitoring.
- [ ] Backup raw payloads to S3/object storage.

## Backend Droid
- [ ] Run `npm run lint && npm run test` before each PR.
- [ ] Update API schema docs / OpenAPI snippet.
- [ ] Verify Prisma migrations against staging DB.
- [ ] Monitor error logs (Sentry) for new endpoints.

## Frontend Droid
- [ ] Lint & unit tests (`npm run lint`, `npm run test`).
- [ ] Visual regression/Storybook checks (if available).
- [ ] Accessibility quick audit (keyboard navigation + color contrast).
- [ ] Update component docs/Storybook notes.

## Data Viz Droid
- [ ] Performance profiling for large datasets (>10k points).
- [ ] Cross-browser render checks (Chrome, Firefox, Safari).
- [ ] Accessibility labels for charts (ARIA, tooltips).
- [ ] Sync visualization utilities with Frontend team.

## Fullstack Droid
- [ ] CI pipeline runs (lint/test/build) on all active branches.
- [ ] Secrets rotation and env var validation.
- [ ] Alert delivery smoke tests (Push/Email) weekly.
- [ ] Changelog & release notes updates.

## UX Design Droid
- [ ] Figma component tokens aligned with design system.
- [ ] Accessibility review (WCAG AA) per flow.
- [ ] Prototyping link validation (no broken share links).
- [ ] Dev handoff packages (annotation, assets export).

## Product Ops / Coordination
