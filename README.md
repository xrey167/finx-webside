# FinX Trading Platform

Modern Trading & Portfolio Management Platform with **Dark Space Purple Theme**.

## ?? Vision
TradingView-like platform with interactive economic world maps and advanced trading features.

## ?? Tech Stack
- Frontend: Next.js 14 + TypeScript + Tailwind CSS
- Backend: Node.js + PostgreSQL + Redis
- Charts: TradingView Library + D3.js
- Real-time: Socket.io WebSockets
